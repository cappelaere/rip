Compression
===========

**As** a web service  
**I want to**	compress large outputs  
**So that I can** reduce bandwidth consumption  
		
> **rationale** This allows reduced bandwidth consumption.

Acceptance Criteria
-------------------

**Given**	large content  
**When**	I compress the content using gzip  
**Then**  	I can reduce bandwidth
